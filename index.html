U1 Argus Remote RFID

Fast browser-based installer for the V1.0 release of the ESP32-C3 Super Mini + PN532 OpenSpool remote RFID reader for Snapmaker U1.
Schneller browserbasierter Installer fuer das V1.0-Release des ESP32-C3 Super Mini + PN532 OpenSpool Remote RFID Readers fuer den Snapmaker U1.
One firmware, one job, one clean setup. Connect your ESP32-C3 by USB, click Install, and flash the current public V1.0 release directly from the browser.
Eine Firmware, eine Aufgabe, ein sauberer Setup-Weg. ESP32-C3 per USB anschliessen, auf Install klicken und das aktuelle oeffentliche V1.0-Release direkt im Browser flashen.

Aktuelles oeffentliches Release

V1.0

ESP32-C3 Super Mini
  • Uses the prepared single-version installer manifest for V1.0
  • Flashes the firmware image from firmware/V1.0/U1_Argus_Remote_RFID_V1_0.ino.merged.bin
  • Best used with Chrome or Edge and a data-capable USB cable
  • Verwendet das vorbereitete Single-Version-Manifest fuer V1.0
  • Flasht das Firmware-Image aus firmware/V1.0/U1_Argus_Remote_RFID_V1_0.ino.merged.bin
  • Am besten mit Chrome oder Edge und einem datenfaehigen USB-Kabel verwenden

What You Need Before Flashing

Was Du Vor Dem Flashen Brauchst

  • ESP32-C3 Super Mini
  • PN532 in HSU mode
  • 4 connection wires
  • Snapmaker U1 with paxx12 Extended Firmware and external filament detection enabled
  • ESP32-C3 Super Mini
  • PN532 im HSU-Modus
  • 4 Verbindungskabel
  • Snapmaker U1 mit paxx12 Extended Firmware und aktivierter externer Filament Detection
After flashing Connect to the hotspot U1-Argus-Setup. The captive portal should open automatically. If not, open http://192.168.4.1.
Nach dem Flashen Mit dem Hotspot U1-Argus-Setup verbinden. Das Captive Portal sollte automatisch starten. Falls nicht, http://192.168.4.1 oeffnen.

If the board is not detected immediately, reconnect it and try again. Some boards may require BOOT to be held while connecting.

Wenn das Board nicht sofort erkannt wird, bitte erneut verbinden und noch einmal versuchen. Bei manchen Boards muss BOOT beim Verbinden gehalten werden.
